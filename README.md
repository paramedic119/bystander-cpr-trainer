# 胸骨圧迫トレーニングアプリ「いのちを救う準備」

救急講習のあとに、自宅で手軽に胸骨圧迫（心臓マッサージ）の質をセルフチェックできるWebアプリです。
スマートフォンのカメラを使って、あなたの姿勢とリズムをAIが優しく評価します。

## 主な機能
- **骨格推定による姿勢チェック**: 肩と手首の角度から、垂直に押せているかを判定します。
- **リズム解析**: 押し込みのテンポを測定し、ガイドライン（100〜120回/分）に合っているか確認します。
- **メトロノーム機能**: 理想的なリズム（110回/分）でガイド音を鳴らします。
- **3段階評価**: ◎、○、△の評価と、前向きなアドバイスを表示します。

## 使い方
1. 床にクッションや枕を置きます。
2. スマートフォンを自分の真横に、全身が映るように立てかけます。
3. アプリのガイドに従って「スタート」を押し、15秒間圧迫を続けます。
4. 終了後、あなたの評価とアドバイスが表示されます。

## 技術仕様
- **Frontend**: HTML5, Vanilla JavaScript, CSS3 (Vite)
- **AI Engine**: MediaPipe Pose (骨格推定)
- **UI Design**: ダークモード・レスポンシブ設計（スマートフォン最適化）
